💸 Expense Tracker (Python + MySQL)

🚀 Overview

A simple console-based Expense Tracker built using Python and MySQL to manage daily expenses.


✨ Features

- Add expenses
- View all expenses
- Delete expenses
- Search by category
- Table format output


🛠️ Tech Stack

- Python
- MySQL


📂 Project Structure

Expense_Tracker/
├── Expenses_miniproject.py
└── README.md


⚙️ Setup

1. Create database and table in MySQL
2. Update database credentials in the Python file
3. Run the program
